# Microsserviços - FIAP

## 📚 Objetivo

Este repositório tem como objetivo reunir os conteúdos estudados, colocando em prática os conceitos fundamentais de desenvolvimento de APIs e arquitetura de microsserviços.

Ao longo das atividades, estão sendo aplicados conhecimentos como:

- Estrutura de projetos com Spring Boot
- Criação de APIs REST
- Métodos HTTP (GET, POST, PUT, DELETE)
- Endpoints
- JSON
- Arquitetura monolítica e microsserviços
- Testes de API com Postman/Insomnia
- Organização de código em camadas

## 📂 Estrutura do repositório

Os projetos e exercícios estão organizados em pastas, de acordo com as aulas e atividades desenvolvidas.

Exemplo de organização:

- `aula01-hello-world`
- `aula02-api-rest`
- `aula03-crud`
- `checkpoint`

## 🛠️ Tecnologias utilizadas

- Java 17
- Spring Boot
- Maven
- IntelliJ IDEA
- Postman / Insomnia
- Git e GitHub
